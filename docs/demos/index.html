PKM Interactive Demos

Four self-contained demo families — drag, tune, diagnose, and watch the physics. No install; they run straight from the page. Every family has a Student and an Instructor mode and an Explore → Challenge → Artifact → Check loop.
1 · Kinematics Explorer →
2-RPR and 3-RPR. Drag the platform; watch inverse/forward kinematics, the Jacobian determinant, manipulability, and singularities update live. Generate workspace, manipulability, and safe-region maps.
Lessons 1.1–1.4, 3.1–3.4 · geometry → IK → FK → workspace → manipulability → singularities
 2 · Hydraulic Explorer →
Cylinder, valve, pump/power, and sensor views. Size areas and asymmetry φ, read the orifice flow law, size the pump, and scale a sensor — with undersize, saturation, and over-pressure faults.
Lessons 2.1–2.4 · cylinder → valve → pump → sensor
 3 · PWM / Control Lab →
The signature control experience. On/off DCV vs PWM vs proportional: duty-cycle sweep, deadband, limit cycles, position control, coordinated tracking, and tuning — with the proportional-vs-on/off overlay.
Lessons 2.5–2.6, 3.5–3.6 · duty → position → coordinated → tuning
 4 · Digital Twin Validation Lab →
Twin vs rig: overlay, time-synchronization, replay, and the Twin Accuracy Report (RMSE, settling, duty, pressure error). Then diagnose the discrepancy — geometry, sensor, deadband, pressure, or timing — from its signature.
Lessons 2.8, 4.1–4.4 · overlay → sync → replay → accuracy → diagnose
