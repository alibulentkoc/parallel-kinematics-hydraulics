# Capstone — End-to-End System Design

The modules teach each piece in isolation. This capstone **assembles them into a
complete machine**, twice — taking a written specification all the way through
kinematics, workspace, actuator and valve sizing, pump and power, control design,
wiring, and validation.

- **[The 2-DOF machine](2dof-design.md)** — positions a point with two hydraulic legs. The midterm machine.
- **[The 3-DOF machine](3dof-design.md)** — positions *and orients* a platform with three legs, and forces you to design around a real singularity. The final-project machine.

Every number on these pages is reproducible in the [Python notebooks](../notebooks/index.md).
Read the 2-DOF page first; the 3-DOF page builds directly on it.
